Quran Browser



This is an experimental project that uses HTML5 WebSQL APIs for locally storing Al Quran, aiming to provide an interface to view and search the holy book. Firefox is not supported at the moment. We will attempt to load the Quran text automatically on first view, after which you will be able to access it offline.

Quran text and translations (coming soon) are courtsey tanzil.net.

We also have a javascript API and a Node.js API for Holy Quran.
